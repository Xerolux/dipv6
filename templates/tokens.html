How to Use Tokens
In UniFi Network Settings:
Server: https://ip.xerolux.net/api/update?ipv6prefix=auto&ipv4=auto&token=YOUR_TOKEN_HERE
Via Command Line:
curl "https://ip.xerolux.net/api/update?ipv6prefix=auto&token=YOUR_TOKEN_HERE"
Via Bash Script:
#!/bin/bash
TOKEN="your-token-here"
curl "https://ip.xerolux.net/api/update?ipv6prefix=auto&ipv4=auto&token=$TOKEN"
Security Tips
  • Each device should have its own token
  • Delete unused tokens
  • Use HTTPS only
  • Rotate tokens regularly
  • Keep tokens secret
